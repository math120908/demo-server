Architecture Overview

Pipeline Data Flow

RAW INPUT → Filter → Generate Entities → Relationships → Virtual Areas → Enrichment → Hotfixes → Standardize → Publish → Validate

Quick Reference

Data Sources

Microsoft Bing Maps geographic data dump in YYYYMMDD format. HDFS working directories under /jobs/msftgeo/.

Output Destinations

Dali views (stz_derived_mp.*), Venice stores (BingGeoEntities, StylizedGeoNames, StandardizedBingGeo, GeoPolygonInfo), Lucene indexes in Artifactory.

Key Concepts

BID (Bing ID) → LID (LinkedIn ID, range 90M–200M) → URN (urn:li:geo:{lid}). Supports 24 locales, market areas, and country clusters.
