Venice Stores

VENICE

BingGeoEntities VENICE

Primary entity lookup store.

Key: LID

Fields: GeoEntity (name, type, adminChain, latLong, etc.)

StylizedGeoNames VENICE

Localized display names.

Key: LID + locale

Fields: stylized name string

StandardizedBingGeo VENICE

Enriched geo records.

Key: LID

Fields: standardized record with city list, member counts

GeoPolygonInfo VENICE

Geographic polygon data.

Key: LID

Fields: polygon boundaries, area

HDFS Commands Reference

Common hdfs-cli commands for inspecting bingo-offline datasets.

# List contents
hdfs-cli -u msftgeo ls /jobs/msftgeo/versions/1/1_2/

# View Avro schema
hdfs-cli -u msftgeo avro-schema dalids:///stz_derived_mp.stz_derived_taxonomy_geo_entities

# Sample records
hdfs-cli -u msftgeo avro-cat dalids:///stz_derived_mp.stz_derived_taxonomy_geo_entities 2>/dev/null | head -5

# Check file size
hdfs-cli -u msftgeo du /jobs/msftgeo/versions/1/1_2/geo_entities
